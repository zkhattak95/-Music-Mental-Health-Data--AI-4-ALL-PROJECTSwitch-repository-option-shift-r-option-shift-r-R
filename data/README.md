# Data folder

This starter contains:

- `music_survey_cleaned.csv` - cleaned music and mental-health survey data.
- `burnout_dashboard_sample.csv` - a compact stratified sample used only for dashboard charts.

The full one-million-row burnout dataset is intentionally excluded because it is too large for ordinary GitHub tracking. Keep the full file locally as:

```text
data/student_burnout_cleaned_full.csv
```

The deployed app uses the saved model and does not require the full dataset.
